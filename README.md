# UI-UX-Experiment1

## Aim:
To understand and demonstrate the fundamentals of UI and UX by exploring their differences, relationships, and practical applications using real-world examples and design tools.

## Algorithm:
1.Study the basic concepts of UI and UX.

2.Compare UI and UX using real-world case studies.

3.Identify how UI and UX work together in the design process.

4.Explore and demonstrate different UI/UX design tools.
